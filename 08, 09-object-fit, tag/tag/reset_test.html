html 주요 태그 종류/특징

form 태그 바로가기

ul 태그: 순서 없는 목록 태그(unordered list)

순서 없는 항목들의 목록을 표시할 때 사용하는 태그

각 목록(리스트)는 li태그로 감싸준다

  • ul 태그 작계 자식인 li 태그
  • ul 태그 작계 자식인 li 태그
  • ul 태그 작계 자식인 li 태그
  • ul 태그 작계 자식인 li 태그

ol 태그: 순서 있는 목록 태그(ordered list)

순서 있는 항목들의 목록을 표시할 때 사용하는 태그

각 목록(리스트)는 li태그로 감싸준다

  1. ol 태그 작계 자식인 li 태그
  2. ol 태그 작계 자식인 li 태그
  3. ol 태그 작계 자식인 li 태그
  4. ol 태그 작계 자식인 li 태그

dl 태그: 용어 설명 목록 태그(definition list)

용어 명칭과 용어에 대한 설명을 쌍으로 묶어 표현해 정의 목록을 만들 때 사용하는 태그

Definition Term 용어 명칭
Definition Description 용어에 대한 설명
용어2
용어2에 대한 설명

video 태그

웹 페이지에 비디오 파일을 삽입하고 재생할 때 사용하는 태그

하단 여백 테스트

iframe 태그

외부 콘텐프를 '프레임'으로 임베딩(삽입)하는 태그

audio 태그

오디오 파일을 삽입에 재생할 수 있는 태그

table 태그

표(데이터 테이블)을 생성하는 태그

표 제목
셀 제목1 셀 제목2
셀 내용1 셀 내용2
셀 내용3
셀 하단

select 태그

'드롭다운 목록(선택 상자)'을 만들 때 사용하는 태그

datalist 태그

'입력 필드' + '드롭 다운' 목록 태그

form 태그

회원가입이나 로그인, 설문조사 페이지 등 '사용자 입력'을 받는 구조를 만들 때 사용

legend 태그: 필드 그룹의 제목
